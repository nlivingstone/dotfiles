## Install
```
git clone git@github.com:nlivingstone/dotfiles.git
cd dotfiles
ln -s nvim ~/.config/nvim
ln -s neovide ~/.config/neovide
ln -s starship.toml ~/.config/starship.toml
ln -s zsh/.zshrc ~/.zshrc
```
